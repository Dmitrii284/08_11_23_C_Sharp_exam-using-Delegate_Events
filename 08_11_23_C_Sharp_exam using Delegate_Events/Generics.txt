Задача: Создать приложение для управления коллекцией автомобилей.

Описание:
Требуется разработать консольное приложение на языке C#, которое позволит пользователю создавать, 
редактировать и удалять записи об автомобилях. У каждого автомобиля должна быть следующая информация:
марка, модель, год выпуска, цвет и цена.

Требования:

Создайте класс Car с нужными свойствами для хранения информации об автомобиле.
Разработайте меню, которое позволит пользователю выбирать операции: добавление нового автомобиля,
редактирование существующего автомобиля, удаление автомобиля или просмотр списка всех автомобилей.
Используйте коллекцию (например, List) для хранения всех автомобилей.
При добавлении нового автомобиля запрашивайте у пользователя необходимую информацию.
Реализуйте функциональность поиска автомобилей по следующим критериям:
По марке автомобиля.
По цвету автомобиля.
По диапазону цен.
Примечания:

Для выполнения операций поиска используйте методы LINQ, такие как Where, Select, и т.д.
Обратите внимание на использование ООП-принципов при проектировании классов и методов.
